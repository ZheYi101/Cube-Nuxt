# 精弘图床
Cube (存储立方) 是用来便于管理精弘开发用的静态文件(主要为图片)的项目
对应后端[后端](https://github.com/zjutjh/Cube-Go)

# 技术栈
`Nuxt`作为框架
配置zjutjh的eslint规范[lint](https://github.com/zjutjh/eslint-config)

用git-hook配置了禁止push到main仓库 使用请先在对应的分支里写 然后提pr
